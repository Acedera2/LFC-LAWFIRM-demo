Demo assets (screenshots, GIFs)

This folder is used to store demo screenshots and GIFs generated by Playwright or the demo scripts.

Commands

- Capture screenshots (Playwright script):

```bash
npm --prefix client install
npx playwright install
npm --prefix client run e2e:screenshots
```

- Convert Playwright videos to GIFs (example using ffmpeg):

```bash
ffmpeg -i client/test-results/playwright-video.webm -r 15 -vf "scale=720:-1:flags=lanczos" -loop 0 client/demo/gifs/demo.gif
```

- Where files appear:
  - Screenshots: `client/demo/*.png`
  - GIFs: `client/demo/gifs/*.gif`

CI: The `.github/workflows/e2e.yml` will convert recorded videos to GIFs and upload them as artifacts automatically.
